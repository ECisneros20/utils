pj_utils.general.log module

pj_utils.general.log.agregar_log_bd(mensaje: str, credenciales: dict = {}) → None[fuente]

Copia el último mensaje del archivo log y lo agrega a la tabla log de PostgreSQL. Para cada nuevo proyecto es necesario cambiar las variables tabla y columnas, según las credenciales que se usarán aparte de las siguientes:

“marca_tiempo - modulo - nivel - mensaje”

Parámetros:

  • mensaje (str) – _description_

  • credenciales (dict, optional) – Datos a registrar en el log. Defaults to {}.

pj_utils.general.log.setup_logging() → None[fuente]

Setea el mensaje de logging para todos los scripts del pj_utils y también del proyecto
